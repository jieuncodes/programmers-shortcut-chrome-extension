  <h2>단축키</h2>
  <table>
    <thead>
      <tr>
        <th>단축키</th>
        <th>설명</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>Cmd/Ctrl + Shit + U</td>
        <td>에디터에서 코드를 실행합니다.</td>
      </tr>
      <tr>
        <td>Cmd/Ctrl + Shift + S</td>

      </tr>
    </tbody>
  </table>

  <h2>설치</h2>
  <ol>
    <li>이 저장소에서 코드를 내려받습니다.</li>

    <li>오른쪽 상단의 토글 스위치를 클릭하여 개발자 모드를 활성화합니다.</li>
    <li>"압축 해제된 확장 프로그램을 로드합니다" 버튼을 클릭하고 다운로드한 폴더를 선택합니다.</li>
